# punch
time tracker. punch in

## Punch in
```
$ punch in something
Initializing ~/.punch.db
Punching into something
```

## Punch out
```
$ punch out
Punched out from 'something' after 2 hours
```

## List 
```
$ punch list something
something
  Date: 2023-11-04
  Duration: 2 hours
  Billed: No
Total: 2 hours
```
